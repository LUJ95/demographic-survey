# 프로젝트 요구사항 정의서

주제 : 인구이동 데이터와 부동산 가격 예측(도시 양극화 분석)

1. 프로젝트 개요 및 목적

 - 목적 : 부동산 가격 예측 알고리즘을 개발하여 고객의 부동산 거래에 도움을 제공하는 것을 목표로 함.
 - 개요 : 
	A. 웹사이트 혹은 어플리케이션은 고객이 지역을 선택하면, 해당 지역의 실거래지수 등락을 예측하여 제공
	B. 부동산 매물 API -> 매물정보 읽고 매물의 미래가격 예측(현실적으로 쉽지 않음)
	https://brunch.co.kr/@clay1987/315

2. 분석 계획

	플랜 A:
	
		# ROW DATA :
		 
		 - 종속 변수(Y) : 지역별 아파트 매매 실거래지수 데이터
		 - 독립 변수(X) : 지지율, PIR지수_전국, 매매수급동향, 소비자물가지수, 종합부동산세_세율_개인, 아파트매매거래량, 주택담보대출
			
		1) 데이터 수집
		 
		 1. 크롤링
		 - 통계청 API 이용하여 파이썬으로 해당 데이터 크롤링을 이용하여 데이터 수집
		 2. 직접 수집
		 - 크롤링이 불가능한 경우 엑셀 파일 다운로드 및 직접 데이터 입력 과정을 거쳐 데이터 수집
		 
		 
		2) 전처리
		
		 1. 데이터 정제
		 - 데이터의 지역 범위는 전국 데이터만 사용
		 2. 데이터 전처리
		 - '시점' 열을 기준열으로 해서 데이터 프레임 결합
		 3. 스케일링 적용
		 - 원본, MinMax스케일링, Robust스케일링, Standard스케일링 등 적용
		 4. 주성분분석 시행
		 - 차원축소, 주성분분석 시행
		 
		 3) 머신러닝 or 딥러닝
